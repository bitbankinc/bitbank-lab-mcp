# チャートを表示する

本サーバーは「ローソク足に指標を重ねたチャート」や「板の深度チャート」を表示できます。描画には **2つのパス** があり、使っているクライアントに応じて自動的に適切な方が選ばれます。

{% hint style="info" %}
**原則**: クライアント側で描画できるなら **Visualizer 用データ（`prepare_*`）** を第一選択に。描画できない環境やファイル保存が必要な場合に **SVG 生成（`render_*`）** へフォールバックします。
{% endhint %}

## パス1：Visualizer 描画（会話内インライン表示）

Claude.ai の Visualizer など、クライアント側で描画できる環境向けに、コンパクトな整形データを返します。LLM も数値を直接参照できるため、「この価格帯に買いが厚い」といった言及も可能です。

| ツール | 用途 |
| --- | --- |
| `prepare_chart_data` | ローソク足＋指標の時系列データ（全指標は計算・シフト適用済み） |
| `prepare_depth_data` | 板の深度チャート用の累積 volume 階段データ（mid・spread・band 集計付き） |

## パス2：サーバー側 SVG 生成（ファイル保存・非 Visualizer 環境）

クライアント側で描画できない場合や、PNG/SVG でファイル保存したい場合のフォールバックです。

| ツール | 用途 |
| --- | --- |
| `render_chart_svg` | メインチャート（ローソク足/ライン + SMA/EMA/BB/一目）+ サブパネル（MACD / RSI / Volume） |
| `render_depth_svg` | 板の深度チャート（累積 bid/ask カーブ） |
| `render_candle_pattern_diagram` | ローソク足パターンの教育用図解 |

## Claude でアーティファクトとして表示するコツ

MCP クライアント（Claude）では、チャートを**アーティファクト**として表示するようお願いするときれいに出ます。うまく表示されない場合は、次の一文を添えてください。

> identifier と title を追加して、アーティファクトとして表示して

{% hint style="warning" %}
チャートは必ずツールが返した `data.svg` をそのまま表示します。LLM が自前で描画コードを書くことはありません（自前描画は禁止）。
{% endhint %}

## オーバーレイの指定

既定の描画は **ローソク足のみ** です。ボリンジャーバンドや一目均衡表などのオーバーレイは、明示的に指定したときだけ追加されます（例: BB は `--bb-mode=default` 指定時に ±2σ）。
