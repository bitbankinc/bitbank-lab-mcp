# 会話レシピ集

ツール名を覚える必要はありません。やりたいことを自然な言葉で話しかければ、適切なツールが選ばれます。ここでは目的別に「聞き方の型」を紹介します。

## まず全体感をつかむ

> 今、BTC は買いですか？

`analyze_market_signal` が総合スコア（-100〜+100）と寄与度・根拠を返します。分析の起点に最適です。

> BTC の今の市場状況を分析して、根拠と寄与度も教えて。

## 注目銘柄を探す（スクリーニング）

> 今注目のコインは？

`get_tickers_jpy` が出来高・24h 変化率でランキング表示します。

> 直近 1 週間でテクニカル的に上向きの仮想通貨を3つ教えて。
> 直近で MACD クロスした銘柄は？

`detect_macd_cross` を `pair` 省略で呼ぶと複数ペアを横断スクリーニングします。

## 1銘柄を深掘りする

> 主要指標で BTC を分析して。

`analyze_indicators` が RSI / MACD / BB / 一目 / SMA を一括取得して総合分析します。

> BTC のサポレジを分析して。
> BTC のフロー分析をして（買い・売りどちらが優勢？）。

## 値動きを可視化する

> ここ 30 日のボラ推移を見たい。

`get_volatility_metrics` ＋ チャート描画でボラティリティの推移を表示します。

> BTC の日足チャートにボリンジャーバンドを重ねて表示して。

可視化の仕組みは [チャートを表示する](charts.md) を参照してください。

## 朝のルーティンに

> おはようレポートを出して。

寝ている間の相場変動を HTML ダッシュボードで素早くキャッチアップできます（Claude Desktop 推奨）。
